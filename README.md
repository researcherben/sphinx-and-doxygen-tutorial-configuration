# sphinx and doxygen configuration for Python and C++

This repo shows how to build Sphinx documentation that leverages documentation generated by Doxygen from C++. `breathe` is the bridge from Doxygen's XML to Sphinx. 
* <https://www.doxygen.nl/manual/index.html>
* <https://www.sphinx-doc.org/en/master/>
* <https://breathe.readthedocs.io/en/latest/>

This repo uses Docker for containerization of dependencies and for a complete capture of the build process.
